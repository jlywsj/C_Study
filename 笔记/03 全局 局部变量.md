### 全局变量

​	全局变量：定义在代码块外部的变量

### 局部变量 

​	 局部变量：定义在代码块内部的变量

 1.当局部变量与全局变量名相同时，局部变量优先
 2.局部变量不能在被定义的代码块外使用
 3.全局变量可以在程序的任意代码块中使用

### 变量作用域

​	局部变量的作用域师变量所在的局部范围
​	全局变量作用域是整个工程

### 变量的生命周期

​	变量的生命周期是指变量的创建到变量的销毁之间的一个时间段
		1. 局部变量的生命周期：进入作用域生命周期开始，出作用域 生命周期结束
		1. 全局变量的生命周期：整个程序的生命周期

### extern关键字

​	extern关键字用来声明外部符号，声明外部文件中定义的全局变量，

​	int main(){
​		{
​			int a = 10;
​			printf("a = %d", a);	// 10
​		}
​		
​		printf("%d",a);   // error
​		return 0;
​	}
